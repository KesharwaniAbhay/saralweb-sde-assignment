# How to Run This NodeJS Assignment

Follow these steps to run the `mergeTimeRanges` module.

## 1. Install Node.js

Download and install Node.js from:
[https://nodejs.org/](https://nodejs.org/)

Verify installation:

```bash
node -v
npm -v
```

## 2. Project Structure

Make sure your folder looks like this:

```
saralweb-sde-assignment/
├─ my-code.js
└─ test.js
```

## 3. Run Test File

Open terminal and navigate to the project folder:

```bash
cd saralweb-sde-assignment
```

Run the code:

```bash
node test.js
```

## 4. Output

You should see merged ranges printed in the terminal.

## Done ✅

Your NodeJS assignment is running successfully.
